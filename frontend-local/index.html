How Search Works

Here's how the search works:

  • We use LLMs to understand the query and rephrase it for better results
  • We query Overture data for administrative boundaries
  • We use LLMs to rank the results based on relevance

Examples:

  • Handles typos: "San fransico" → "San Francisco"
  • Understands context: "London not in the UK but in Canada" → "London, Ontario"
  • Processes verbose queries: "the great city of New York" → "New York City"

This is a very early prototype, so the results might not always be accurate.
